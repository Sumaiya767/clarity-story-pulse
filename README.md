# StoryPulse
A decentralized story-sharing platform that combines audio narration with visual media on the Stacks blockchain.

## Features
- Create and publish stories with audio and image metadata
- Like and tip storytellers
- Follow favorite creators
- Story ownership tracked via NFTs

## Contracts
- `story-pulse.clar`: Core contract handling story creation and management
- `story-token.clar`: NFT contract for story ownership
- `creator-registry.clar`: Contract managing creator profiles and following system
